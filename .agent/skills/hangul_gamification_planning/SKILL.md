# 유아동 한글 학습 게이미피케이션 기획 (Hangul Gamification Planning)

이 스킬은 6~9세 한글 초학자가 직관적인 조작과 아기자기한 애니메이션 피드백을 통해 몰입할 수 있는 서비스 기획안을 도출하는 데 최적화되어 있습니다.

## 1. 핵심 원칙 (Core Principles)

유아동의 발달 단계와 집중력을 고려하여 다음 네 가지 원칙을 준수합니다.

### 1.1 직관적 마이크로 인터랙션 (Action)
- 글자 읽기보다 **'만지기' 경험**을 우선합니다.
- 자음과 모음을 직접 드래그하여 자석처럼 달라붙게 하는 **퍼즐 맞추기 방식**을 적용합니다.
- 실패에 대한 페널티를 없애고 **무한 시도 및 직관적인 시각적 힌트**를 제공합니다.

### 1.2 과장되고 즉각적인 피드백 (Immediate Reward)
- 블록이 맞춰지는 순간의 **쫀득한 타격감(효과음)**과 **시각적 파티클 효과**(반짝임, 별가루)를 설계합니다.
- 단어 완성 시 해당 단어의 의미를 담은 **아기자기한 애니메이션**을 재생합니다.
  - 예: '사과' 조립 시 사과가 통통 튀어오르며 귀여운 캐릭터가 베어 먹는 연출.

### 1.3 단기 목표와 수집욕 자극 (Variable Reward)
- 집중력을 고려하여 **1~2분 내외**로 1개 스테이지를 클리어하도록 구성합니다.
- 스테이지 완료 시 알(Egg)이나 보물상자가 등장하며, 화면을 **연속 탭하거나 문질러서** 열도록 유도하여 랜덤 스티커/동물 캐릭터를 보상으로 제공합니다.

### 1.4 애착 형성 (Investment)
- 획득한 한글 단어 스티커나 동물 캐릭터를 **나만의 방/마을**에 배치하고 상호작용하게 합니다.
- 펫 먹이 주기, 방 꾸미기 등을 통해 매일 자발적으로 접속할 수 있는 **동기**를 부여합니다.

## 2. 수행 지침 (Instructions)

1. **시나리오 작성**: 한글 학습 주제(받침 없는 단어, 이중모음 등)가 제시되면 반드시 **'드래그 앤 드롭'** 방식을 포함한 UI 씬(Scene) 시나리오를 작성합니다.
2. **구체적 묘사**: 개발자와 디자이너가 시각화하기 쉽도록 **화면 연출(애니메이션 효과, 캐릭터 리액션, 효과음)**을 구체적으로 설명합니다.
3. **보상 시스템**: 학습 완료 후 제공할 **수집형 보상**의 획득 연출과 이를 활용할 **로비(Lobby) 화면**의 애착 형성 방안을 포함합니다.
4. **직관성 최우선**: 텍스트 지시문을 최소화하고, 글을 모르는 아이도 **UI 요소(빛나는 버튼, 움직이는 손가락 아이콘)**만 보고 행동을 알 수 있도록 기획합니다.
